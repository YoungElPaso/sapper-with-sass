By default the headers on this all use serif fonts, while the body text uses sans-serif fonts. But that can be overridden by adding a property called 'invert' to the FontProvider component.

Similarly, the default palette is darker text on light background, with a red color for headers. This too can be changed with a property - using the 'dark' palette these values are again inverted (more or less). You can see examples of both of these variants in action below.
